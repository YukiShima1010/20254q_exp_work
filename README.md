# プロダクト開発: 10年後のZEN Study

## コンセプト

・色々学べる
- 楽しく、幅広い知識を身につけ、わからないをわかるに変えるZENStudy
- 年代を超えて学べるZEN Study
- 数多の教材にアクセスしやすく

## 担当別ワーク

### 概要・ターゲット・ビジネスモデル
- ZEN Studyは小学生からシニア層だけでなく、国境や人種を超えて世界中の人々に多種多様で受け放題の授業・講義をオンラインで提供するサービスです。
- 特徴
  - 小学校の教材からプログラミング、フィットネス、大学レベルの高度な授業まで受け放題
  - 定額制かつ非常に安価
  - AIによる学習サポート
- ターゲット
  - 小学生〜シニア層までの幅広い年代
- ビジネスモデル
  - 現代の学び方を変える(＝体験型にする)
  - SDGsの2つの目標を達成する
    - 3番「すべての人に健康と福祉を」
    - 4番「だれもが公平に、良い教育を受けられるように、また一生に渡って学習できる機会を広めよう」
  - 上記二つのZEN Studyの価値を利用者に届け、サブスクリプションとして月々定額の受講料を享受する。

### 画面の構成・UI
- 教材の一覧を確認したい時上にスクロールしないと確認できないため、教材一覧(コースを探す)をフローティングバナーにする
- 教材一覧の一番上に検索ボックスを設置する
- アカウントから言語を変更
- 言語変更の際に教材の画像、図を翻訳したものに差し替え

### 機能一覧
- 解説機能(文章やイラスト、図などを使い解説)
- 教材検索機能(授業選択の効率化)
- AR機能(持ち運び化)
- タイムトラベル機能
　-昔の歴史・文化を体験し、学ぶ機能
　-学習支援機能(現在の学習履歴を元に未来の理解度やテストの点数等を予想し、スケージュールを立てる機能)
