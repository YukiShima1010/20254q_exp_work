# プロダクト開発: 10年後のZEN Study

## コンセプト

・色々学べる
- 楽しく、幅広い知識を身につけ、わからないをわかるに変えるZENStudy
- 年代を超えて学べるZEN Study
- 数多の教材にアクセスしやすく

## 担当別ワーク

### 概要・ターゲット・ビジネスモデル
- ZEN Studyは小学生からシニア層まで、幅広い年代の人が多種多様で受け放題の授業・講義をオンラインで受けられるサービスです。
- 特徴
  - 小学校の教材からプログラミング、フィットネス、大学レベルの高度な授業まで受け放題
  - 定額制かつ非常に安価
  - AIによる学習サポート
- ターゲット
  - 小学生〜シニア層までの幅広い年代
- ビジネスモデル
  - カキカキ中

### 画面の構成・UI
- 教材の一覧を確認したい時上にスクロールしないと確認できないため、教材一覧(コースを探す)をフローティングバナーにする
- 教材一覧の一番上に検索ボックスを設置する

### 機能一覧
- 解説機能(文章やイラスト、図などを使い解説)
- 教材検索機能(授業選択の効率化)
